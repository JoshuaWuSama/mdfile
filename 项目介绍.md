### OpenClaw Agent示例（伪代码）

```python
class StudentAnalysisAgent:

    def run(self, student_id):

        answers = get_student_answers(student_id)

        knowledge_stats = {}

        for answer in answers:

            knowledge = answer.knowledge_id

            if knowledge not in knowledge_stats:
                knowledge_stats[knowledge] = []

            knowledge_stats[knowledge].append(answer.is_correct)

        mastery = {}

        for k,v in knowledge_stats.items():
            mastery[k] = sum(v)/len(v)

        update_student_model(student_id, mastery)

        return mastery
```

---

## BKT 模型

知识状态：

```
Learned
Not Learned
```

公式：

```
P(Ln|Correct) =
(P(Ln−1) × (1 − S)) /
((P(Ln−1) × (1 − S)) + (1 − P(Ln−1)) × G)
```

参数：

```
P(L)  初始掌握
S     slip（会但做错）
G     guess（不会但猜对）
```

---

## 简化版本

实际工程可以用：

```
mastery =
正确率 × 难度权重 × 时间权重
```

示例：

```
欧姆定律

正确率 0.5
难度权重 1.2

mastery = 0.6
```
